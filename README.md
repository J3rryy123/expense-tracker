## ✨ Features
- Categorize and manage expenses
- Interactive charts with Chart.js
- Filters and export functions

## 🛠️ Technologies
- HTML5, CSS3, JavaScript ES6+
- Chart.js for data visualization
- CSS Grid & Flexbox
- Local Storage API
